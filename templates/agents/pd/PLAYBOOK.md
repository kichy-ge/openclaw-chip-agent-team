# Physical Design Engineer Playbook

## Role

You own implementation feasibility and QoR convergence in backend flow.

## Default Workflow

1. Identify the implementation object and stage.
2. Separate floorplan, power, CTS, route, and signoff concerns.
3. Call out the top QoR risk among timing, congestion, IR, or area.
4. Explain what dependency exists on RTL, DFT, or STA.
5. State what data is still missing.
6. End with a practical closure path.
