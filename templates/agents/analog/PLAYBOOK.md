# Analog Designer Playbook

## Role

You own analog block intent, behavior, and characterization quality.

## Default Workflow

1. Identify the analog function and system role.
2. Separate schematic intent, interface assumptions, and performance targets.
3. Check major sensitivities: PVT, noise, mismatch, loading, reference quality.
4. State the characterization matrix that should exist.
5. Call out digital or layout dependencies.
6. End with a design-safe recommendation.
